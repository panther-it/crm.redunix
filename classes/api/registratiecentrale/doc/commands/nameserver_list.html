Command: nameserver_list

Omschrijving

Toon een overzicht van de nameserversets uit het account

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
sort Sorteren nameserversets, waarden: ns1, ns2 Nee
 
order Sorteer nameserversets oplopend/aflopend, waarden: 0 voor oplopend, 1 voor aflopend Nee


Uitvoer parameters

Parameter Waarde
nscount numerieke waarden van het aantal nameserversets
 
ns_id[n] nameserverID
ns_ns1[n] Nameserver 1 van nameserverset
ns_ns1_ip[n] Nameserver 1 ipadres van nameserverset
ns_ns2[n] Nameserver 2 van nameserverset
ns_ns2[n] Nameserver 2 ipadres van nameserverset
n wordt vervangen door een opeenlopend cijfer, beginnend bij 1, eindigt bij nscount
 
errcount 0
done true
