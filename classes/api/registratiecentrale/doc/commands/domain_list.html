Command: domain_list

Omschrijving

Toon een overzicht van de domeinnamen uit het account

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
tld Toon domeinnamen van TLD Nee
sort Sorteren domeinnamen, waarden: domein/registrant/admin/tech/verloopdatum/status Nee
order Sorteer domeinnamen oplopend/aflopend, waarden: 0 voor oplopend, 1 voor aflopend Nee
begin Toon domeinnamen beginnend met letter van alfabet, waarden: a,b,c….x,y,z of 0-9 Nee


Uitvoer parameters

Parameter Waarde
domeincount nummerieke waarde van het aantal domeinnamen
 
domein[n] Domeinnaam
registrant[n] Naam van registrant
admin[n] Naam van admin. contact
tech[n] Naam van tech. contact
verloopdatum[n] Verloopdatum van domeinnaam in formaat: dd-mm-jjjj
status[n] Status van domeinnaam
n wordt vervangen door een opeenlopend cijfer, beginnend bij 1, eindigt bij domaincount
 
errcount 0
done true
